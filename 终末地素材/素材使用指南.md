## 三、设计规范参考

### 3.1 配色方案

从SVG素材提取的主要颜色：

| 颜色名称 | HEX值 | 用途 |
|---------|-------|------|
| 深灰背景 | `#363535` / `#2F2F2F` | 主背景、容器 |
| 浅灰图标 | `#D3D3D3` / `#B7B7B7` | 图标填充 |
| 纯白 | `#FFFFFF` | 文字、高亮 |
| 亮黄 | `#FFFF0F` | 强调、能量相关 |
| 中灰 | `#646464` / `#434343` | 次要元素 |

### 3.2 字体使用建议

```css
/* 标题字体 */
font-family: 'Gilroy-Black', 'HarmonyOS Sans SC Black', sans-serif;

/* 副标题字体 */
font-family: 'Gilroy-Bold', 'HarmonyOS Sans SC Bold', sans-serif;

/* 正文字体 */
font-family: 'Gilroy-Medium', 'HarmonyOS Sans SC Medium', sans-serif;
```

### 3.3 文件命名规范

素材采用 `[名称].[hash].[扩展名]` 的命名格式：
- 名称：描述性英文名
- Hash：8位内容哈希（用于缓存控制）
- 扩展名：文件类型

示例：
- `avatar.a68a95.png`
- `modal-bg-large.c83e18.png`

## 四、使用建议

### 4.1 素材选择指南

| 需求场景 | 推荐素材来源 |
|---------|-------------|
| UI图标系统 | 临行事项 |
| 角色表情动画 | 电力满格 |
| 背景/装饰 | 镌宇铭星、集结出发 |
| 弹窗/模态框 | 电力满格 |
| 抽奖系统 | 临行事项、电力满格 |
| 音效反馈 | 电力满格 (`UI_*.mp3`) |

### 4.2 格式选择

| 场景 | 推荐格式 | 原因 |
|------|---------|------|
| 图标/装饰 | SVG | 可缩放、文件小 |
| 照片/背景 | JPG | 压缩率高 |
| 透明图 | PNG | 支持透明通道 |
| 现代浏览器 | WebP | 更好的压缩比 |
| 音效 | MP3 | 兼容性好、体积小 |
| 字体 | WOFF2 | 最佳压缩 |

### 4.3 性能优化建议

1. **字体加载**：优先使用 WOFF2 格式，配合 `font-display: swap`
2. **图片懒加载**：对非首屏图片使用 `loading="lazy"`
3. **音频预加载**：对关键音效使用 `<audio preload="auto">`
4. **精灵图**：Pixi精灵图已优化，可直接使用
