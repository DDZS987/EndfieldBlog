## 四、使用建议

### 4.1 按需求场景选素材

| 需求场景 | 推荐来源 |
|---------|---------|
| SVG 图标系统 | 临行事项（状态图标、功能图标、装饰 SVG） |
| 角色表情/对话动画 | 电力满格（Perlica/Yvonne 8 张立绘+纹理贴图） |
| 深色星空背景 | 镌宇铭星（与项目现有风格最契合） |
| 飞船/工业场景插画 | 镌宇铭星 |
| 弹窗/模态框套件 | 电力满格（small/medium/large/ex 4 套） |
| 任务进度/解锁 UI | 临行事项（3 组 lock/open 状态图） |
| 抽奖系统 | 临行事项 + 电力满格 |
| 音效反馈 | 电力满格（`UI_Click/Coin/Lottery/POP`） |
| KV 大图/氛围图 | 集结出发（三人组宇宙 KV） |

### 4.2 格式选择

| 场景 | 推荐格式 | 原因 |
|------|---------|------|
| 图标/装饰 | SVG | 可缩放、文件小 |
| 照片/背景 | JPG | 压缩率高 |
| 透明图层 | PNG | 支持透明通道 |
| 现代浏览器 | WebP | 更好压缩比（镌宇铭星全用此格式） |
| 音效 | MP3 | 兼容性好、体积小 |
| 字体优先 | WOFF2 > WOFF > TTF | 压缩率递减 |

### 4.3 性能优化

1. **字体**：优先 WOFF2，配合 `font-display: swap`
2. **图片懒加载**：非首屏图片加 `loading="lazy"`
3. **音频预加载**：关键音效用 `<audio preload="auto">`
4. **精灵图**：Pixi 精灵图（临行事项）已优化，配合 JSON 配置直接使用
